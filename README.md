# Real-World-Analysis
Using “electricity demand.csv” power consumption data, as part of a school assignment, I used Python to calculate the 3-Month Simple Moving Average, 3-Month Weighted Moving Average, Exponential Smoothing (using two different constants), &amp; Linear Regression. Then, I created a table that displayed the error metrics for each method. 

The overall goal of the project was to predict the power consumption needs for the upcoming 6 months based on previous usage data.
